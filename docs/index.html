socialcard

Generate beautiful social card images (OG, Twitter, GitHub) with a fluent builder API. One dependency: Pillow.

pip install socialcard
PyPI ClawHub Source
from socialcard import SocialCard

SocialCard("og")
  .title("My Project")
  .subtitle("A cool tool")
  .render("card.png")
